# Clima

O módulo de dados `vibe_core.data.weather` define tipos de dados relacionados a informações meteorológicas e previsões. Ele fornece classes para representar dados climáticos e previsões do Global Forecast System (GFS), facilitando o trabalho com dados relacionados ao clima em vários cenários de análise agrícola e ambiental.
