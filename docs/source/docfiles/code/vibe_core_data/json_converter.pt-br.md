# Conversor JSON

O módulo `vibe_core.data.json_converter` fornece funções utilitárias e classes para trabalhar com serialização e desserialização JSON de dataclasses e modelos Pydantic. Este módulo simplifica o processo de conversão de dataclasses e modelos Pydantic para o formato JSON e vice-versa, agilizando o gerenciamento e a organização de dados no FarmVibes.AI.
