# Produtos

O módulo `vibe_core.data.products` fornece um conjunto de classes projetadas para lidar com dados de produtos de sensoriamento remoto, incluindo metadados de produtos e dados raster como Mapas Digitais de Elevação (DEM), Programa Nacional de Imagens Agrícolas (NAIP), Landsat, Sentinel e mais.
