# Sentinel

O módulo `vibe_core.data.sentinel` estende o módulo de dados principal para lidar com tipos de dados específicos relacionados aos produtos de satélite Sentinel. Ele fornece classes para representar metadados, rasters e sequências de produtos Sentinel, facilitando o trabalho com dados obtidos dos satélites Sentinel-1 e Sentinel-2.

Além de lidar com produtos Sentinel, o módulo também inclui classes para lidar com rasters e sequências SpaceEye, expandindo ainda mais suas capacidades no domínio do processamento e análise de dados de satélite.
