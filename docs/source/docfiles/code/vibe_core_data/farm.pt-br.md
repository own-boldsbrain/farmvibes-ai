# Fazenda

O módulo `vibe_core.data.farm` fornece classes de dados para representar várias operações e práticas agrícolas em uma fazenda, tais como preparo do solo, fertilização, emendas orgânicas e colheitas. Este módulo simplifica o processo de gerenciamento e organização de dados relacionados a fazendas e oferece tipos de dados especializados para integração com o Azure Data Manager for Agriculture (ADMAg).
