# Utilitários

O módulo `vibe_core.data.utils` fornece funções utilitárias e classes para trabalhar com itens STAC (SpatioTemporal Asset Catalog) e convertê-los de e para objetos `BaseVibe`. Este módulo simplifica o processo de gerenciamento e organização dos dados usados em todo o FarmVibes.AI, oferecendo funções utilitárias para serialização e desserialização de itens STAC, conversão de campos e resolução de tipos.
